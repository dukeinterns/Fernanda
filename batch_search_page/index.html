Create DUIDs From CSV

Show Instructions

- The Batch DUID creation tool accepts a csv file. The CSV file should have the following Headers:

- firstName,lastName,dobMonth,dobDay,dobYear,email

- Name fields should be mixed case.

- Birth Day and Month should be two digits e.g. 01, 07, 22

- Birth year should be four digits e.g. 1984

- Selecting "Dry Run" will not actually create any duids and instead provide a "what if" scenario
